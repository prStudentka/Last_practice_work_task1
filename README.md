# Тест-кейсы

## 1 Проверить кнопку минимальной стоимости кредита

- 1 Зайти на сайт mtsbank.ru
- 2 Навести курсор на категорию «Кредиты»
- 3 Выбрать раздел «Кредиты наличными»
- 4 В блоке "Рассчитайте свой кредит" нажать на "20 тыс ₽"
